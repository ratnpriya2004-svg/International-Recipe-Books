# International-Recipe-Books
A responsive web application to explore international recipes from different countries with search and filtering features using HTML, CSS, and JavaScript.

🍽️ Key Highlights

🌍 Explore recipes from 12+ countries
🍲 240+ traditional and modern recipes
🔍 Smart search for countries and recipes
🥗 Filter by cuisine type (Veg / Non-Veg / Dessert / Quick meals)
📱 Fully responsive design for mobile and desktop
🍔 Mobile-friendly hamburger menu
🔐 Admin panel for control features
🌙 Dark/Light theme support
📊 Data export functionality
